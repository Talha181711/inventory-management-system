# Inventory Management System

## Tech Stack

Frontend:

- React
- Tailwind CSS
- Axios

Backend:

- Laravel 12
- MySQL

## Features

- Brand Management
- Model Management
- Item Management
- Pagination
- Validation
- Success/Error Messages
- Delete Confirmation Modal

## Installation

### Backend

composer install

cp .env.example .env

php artisan key:generate

php artisan migrate

php artisan serve

### Frontend

npm install

npm run dev
